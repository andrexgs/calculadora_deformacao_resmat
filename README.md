# 📐 Calculadora de Deformação (ResMat)

Aplicação web simples para cálculo de deformação axial utilizando a fórmula:

δ = (P · L) / (E · A)

## 🚀 Funcionalidades

- Cálculo automático da deformação (δ)
- Suporte a prefixos SI:
  - k (quilo), M (mega), G (giga), m (mili), μ (micro), etc.
- Alternância automática de unidades:
  - `k` → sistema em mm / mm² / kN
  - outros → sistema em cm / cm² / N
- Interface simples e responsiva
- Resultado exibido em milímetros (mm)

## 🧮 Como usar

1. Insira os valores:
   - **P (Carga)** → Ex: `10k`, `500`, `2M`
   - **L (Comprimento)**
   - **E (Módulo de elasticidade)** → Ex: `210G`
   - **A (Área)**
2. O resultado será calculado automaticamente.

- Use vírgula ou ponto decimal
